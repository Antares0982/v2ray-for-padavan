操作概要：

* 复制`AiDisk_a1`里的`xray`文件夹到路由相同目录（相当于复制到外储存设备里）

* 修改`xray/config.json`为你客户端信息（默认gfwlist模式，公益节点均衡负载）

* 添加运行权限（注意路径！）

  `chmod 775 -R /media/AiDisk_a1/xray/`

* 启动命令

  `sh /media/AiDisk_a1/xray/start.sh`

* 停止操作
  `sh /media/AiDisk_a1/xray/stop.sh`

* 自启动：
  在自定义设置-脚本-'在防火墙规则启动后执行下'粘贴

  ```
  sleep 10
  sh /media/AiDisk_a1/xray/start.sh
  ```
